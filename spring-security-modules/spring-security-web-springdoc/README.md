## 运行项目

要运行项目，请使用以下命令：

- `mvn spring-boot:run -Dstart-class=cn.tuyucheng.taketoday.basicauth.SpringBootSpringdocBasicAuth`
- `mvn spring-boot:run -Dstart-class=cn.tuyucheng.taketoday.formlogin.SpringBootSpringdocFormLogin`
