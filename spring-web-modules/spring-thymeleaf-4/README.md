## 构建项目

mvn clean install

## 运行项目

mvn cargo:run

- **note**: starts on port '8082'

使用以下网址访问页面：

- http://localhost:8082/spring-thymeleaf-4/
- http://localhost:8082/spring-thymeleaf-4/addStudent/
- http://localhost:8082/spring-thymeleaf-4/listStudents/

第一个URL是应用程序的主页，主页包含指向第二页和第三页的链接。
